# 云归标准发布包

这个文件夹用于上传到 GitHub Pages。

## 使用方式
在终端执行：

```bash
cd "/Users/admin/Desktop/KOKO-app/云归-标准发布"
./copy_from_source.command
```

执行后，本文件夹会生成以下可发布文件：

- `index.html`
- `styles.css`
- `app.js`
- `张国荣.jpg`
- `default-custom.png`
- `蜡烛.mp4`
- `蜡烛5.mp4`
- `logo.svg`
- `转发.svg`
- `看过.svg`
- `全屏.svg`

然后把整个 `云归-标准发布` 上传到 GitHub 仓库即可。
